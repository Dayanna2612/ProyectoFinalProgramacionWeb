# 🎮 ArcadeX

Aplicación web tipo CRUD para gestionar videojuegos.  
Permite agregar, listar, editar y eliminar videojuegos con una interfaz moderna estilo neon.

---

## 🚀 Tecnologías usadas

### 🔹 Frontend
- HTML
- CSS (estilo neon ✨)
- JavaScript (Vanilla)
- Vite

### 🔹 Backend
- Node.js
- NestJS
- TypeScript

---

## 📦 Instalación

### 1. Clonar el repositorio
git clone <tu-repo>
cd ProyectoFinalProgramacionWeb

---

## ⚙️ Backend

cd backend
npm install
npm run start:dev

Corre en:
http://localhost:3000

Endpoint principal:
http://localhost:3000/videojuegos

---

## 🎨 Frontend

cd frontend
npm install
npm run dev

Corre en:
http://localhost:5173

---

## 📌 Funcionalidades

✔ Listar videojuegos  
✔ Agregar videojuegos  
✔ Editar videojuegos  
✔ Eliminar videojuegos  
✔ Interfaz moderna con tarjetas  
✔ Diseño responsive estilo neon
